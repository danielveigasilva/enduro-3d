# Enduro-3d

## Comandos
<kbd>X</kbd> : Amplia/Recolhe o mapa

<kbd>C</kbd> : Alterna entre a câmera em terceira e primera pessoa

<kbd>R</kbd> : Inicia/Para a gravação de nova tragetória para competidor

<kbd>Z</kbd> : Alterna entre os modos de renderização de objetos


<kbd>&#8593;</kbd> : Acelera gradualmente

<kbd>&#8595;</kbd> : Desacelera gradualmente

<kbd>&#8592;</kbd> : Curva para a Esquerda

<kbd>&#8594;</kbd> : Curva para a Direira
